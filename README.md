# Ender 3 V3 KE - FatBurner Dual 5015 Cooling Upgrade

This repo documents my **FatBurner** dual 5015 part cooling upgrade on the Creality Ender 3 V3 KE.

## Overview
Replaces the stock weak/noisy cooling with two 5015 blower fans + one 4020 hotend fan for significantly better part cooling and lower noise.

**Biggest quality improvement** after basic mods (dryer box, runout sensor relocation, anti-vibration feet).

## Results
- ~11 dB quieter (54 dB → 43 dB)
- Roughly **half as loud** subjectively
- Much better overhangs, bridges, and high-speed printing (especially PLA/PETG)

## Parts to Print
From: https://www.printables.com/model/887597-fatburner-ender-3-v3-ke-dual-5015-fan-cooling-syst

- Base L + Base R
- Duct L + Duct R
- Cover L + Cover R + Cover Top
- 4020 Fan Adapter

**Print Settings:**
- Bases + Ducts: PETG
- Covers: PLA or PETG
