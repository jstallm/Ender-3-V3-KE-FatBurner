# Ender 3 V3 KE - FatBurner Dual 5015 Cooling Upgrade

This repo documents the **FatBurner** dual 5015 part cooling upgrade for the Creality Ender 3 V3 KE.

## Overview
Replaces the stock weak/noisy cooling with two powerful 5015 blower fans + one 4020 hotend fan.

**Biggest quality jump** after basic mods (dryer routing, runout sensor on head, anti-vib feet).

## Results
- ~11 dB quieter (54 dB → 43 dB)
- Roughly **half as loud**
- Much better overhangs, bridges, and high-speed prints

## Parts to Print
From: [FatBurner on Printables](https://www.printables.com/model/887597)

- Base L + Base R
- Duct L + Duct R
- Cover L + Cover R + Cover Top
- 4020 Fan Adapter

**Print Settings:** Bases & Ducts in PETG, Covers in PLA/PETG.

- **M3×20mm Flat Head Screws** (need 8)  
  Search on AliExpress "M3x20 flat head screw" — ~$2–4 for 50pcs

- **JST 1.25mm Connectors** (optional, pack of 20–30)  
  [AliExpress Direct](https://www.aliexpress.com/item/1005006xxxxxxx.html) — ~$3–5 (search "JST 1.25 2pin")

**Recommendation:** Use **AliExpress** for lowest price if you can wait 2-3 weeks. Use **Amazon** for fast shipping.

## Tools Needed
- Soldering iron (or wire strippers + heat shrink)
- 2.5mm hex key
- Small Phillips screwdriver

## Install Notes
- Compatible with runout sensor on head
- Re-run input shaping / ADXL tune after install
